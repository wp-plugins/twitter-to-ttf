Takes a Twitter ID and grabs the latest Tweet and displays it in a TTF

== Description ==

Simple. Asks you for your Twitter ID. Grabs your latest tweet. Renders it in a transparent PNG using a True Type Font.

== Installation ==

1. Upload all files into a twittertottf folder in /wp-content/plugins
2. Activate the plugin in Wordpress Admin.
3. Add widget to sidebar and configure.
